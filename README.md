## Set Up Instructions For Running Website
- Start the Apache and MySQL modules using the **XAMPP** controller.
- Open the **phpMyAdmin** and create a database **"ecommerce"**. 
- Import the **ecommerce.sql** file present in the zip folder.
- Open the htdocs folder in the xampp folder. Copy paste the folder **"ecommerce"**.
- Open the browser (chrome), type *localhost/ecommerce* and you should see the index page of the website.
